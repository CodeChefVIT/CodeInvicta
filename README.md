## Instructions to run
```bash
$ git clone https://github.com/CodeChefVIT/CodeInvicta
$ cd CodeInvicta
$ npm i
# Development build
$ npm start
# Production build
$ npm run build
```

## Environvental variable(s)
- `REACT_APP_RECAPTCHA_KEY`
